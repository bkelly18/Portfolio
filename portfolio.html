PORTFOLIO

  • Decimal to Binary Converter

      • This project is a web application that can take any random decimal number between zero and 255, then convert the number to a binary number.

      If the selected random number is less than zero or greater than 255, an error message will appear.

      Once a new number is selected, the previously provided binary number will be changed for the new number.

  • FizzBuzz Challenge

      • In this assignment, the goal is to develop the FizzBuzz app from scratch and have it determine if a number is divisible by 3, by 5, or by both 3 and 5.

      If the number is divisible by 3, instead of the number, the outcome will show "fizz"; if the figure is divisible by 5, "buzz" is the response; and finally, if a given number is divisible by both 3 and 5, the answer will be "fizzbuzz".

  • YouTube API Project

      • Requirements for this project include building a simple search form which enables users to search YouTube videos. Based on the search results, video thumbnails will display which match the search.

  • Shopping List

      • To complete this project, a shopping list is to be designed with popular items listed, such a grocery items. The list will allow items to be deleted and other items to be added.
